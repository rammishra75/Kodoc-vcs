# Kodoc - Git for Documents

<!-- Updated README for improved documentation and clarity -->
> A minimalist document editor with powerful version control. Like Git, but for your documents.

## 🎯 Overview

Kodoc is a web-based document editor that treats every save as a commit. Never lose your work, revert to any previous version, and maintain a complete history of your document's evolution.

## ✨ Features

### Core Functionality
- **Rich Text Editor** - Clean, distraction-free writing experience with full formatting support
- **Version Control** - Every save creates a new version with optional commit message
- **Version Tree** - Visual timeline of all document versions
- **Time Travel** - Preview any previous version and restore it instantly
- **Document Management** - Create, edit, organize, and manage all your documents
- **User Authentication** - Secure personal workspace for your documents

### Editor Features
- Bold, italic, underline, strikethrough
- Headings (H1, H2, H3)
- Bullet and numbered lists
- Text alignment
- Links
- Code blocks
- Blockquotes

## 🛠️ Tech Stack

### Frontend
- **React** - UI framework

### Backend
- **Node.js** - Runtime

- **MongoDB** - Database

- **bcryptjs** - Password hashing
